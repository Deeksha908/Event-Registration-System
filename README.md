# Event Registration System

A Flask-based Event Registration System developed for the CodeAlpha Backend Development Internship.

## Features
- View available events
- Event details page
- Register for events
- Store registrations in SQLite
- View all registrations
- Delete registrations

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS

## How to Run

1. Install Flask:
pip install flask

2. Run the project:
python app.py

3. Open:
http://127.0.0.1:5000
